📊 Customer Behaviour Analysis Dashboard
📌 Overview

This project focuses on analyzing customer purchasing behavior using Python, SQL, and Power BI. The goal is to extract meaningful insights from raw data, understand customer segments, and visualize trends through an interactive dashboard.

The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and dashboard creation.

📂 Dataset
The dataset contains customer-level transactional data, including:
Customer ID
Gender
Age Group
Purchase Amount
Category & Product Purchased
Review Rating
Subscription Status
Discount Applied
Shipping Type
Previous Purchases

🛠️ Tools & Technologies
Python (Pandas, NumPy) → Data loading, cleaning, EDA
SQL (MySQL) → Data querying and analysis
Power BI → Dashboard creation and visualization
Jupyter Notebook → Development environment
Gamma → Presentation (PPT creation)

⚙️ Steps Performed
1. Data Loading
Loaded dataset using Python (Pandas)
Connected to MySQL database via Jupyter Notebook
2. Data Cleaning
Handled missing values
Standardized categorical values
Removed inconsistencies
3. Exploratory Data Analysis (EDA)
Analyzed distributions (purchase amount, ratings)
Identified trends across categories and demographics
4. SQL Analysis

Performed multiple business queries using MySQL:

Revenue by gender
Discount impact on spending
Top-rated products
Shipping type comparison
Subscription vs non-subscription analysis
Customer segmentation (New, Returning, Loyal)
Product performance by category
Repeat buyers behavior
Revenue contribution by age group

📊 Dashboard

The Power BI dashboard provides a visual summary of key insights:
Total Customers: 3900
Average Purchase Amount: $59.76
Average Review Rating: 3.75
Key Visuals:
Revenue & Sales by Category
Revenue & Sales by Age Group
Subscription Distribution
Interactive Filters:
Gender
Category
Shipping Type
Subscription Status

The dashboard helps stakeholders quickly understand customer trends and make data-driven decisions.

📈 Results & Insights
Clothing category generates the highest revenue and sales
Young adults contribute the most to revenue
Majority of customers are non-subscribers, but subscribers show strong spending patterns
Customers using discounts still spend above average in many cases
Repeat buyers have a higher likelihood of subscribing
Shipping type has a measurable impact on purchase behavior
